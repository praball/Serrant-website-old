Second Project Website

June 8, 2022 by Prabal

Hi everyone, in this project, I've tried to make a fully responsive website with the help of Bootstrap framework. My earlier project waws mainly focused on using HTML, CSS and Javascript, so I decided to learn and make use of different frameworks.
This website shows the user the latest ongoing news in the web-series, TV Shows and Anime/Shounen world.

I have duly taken my references from a number of real-time websites around the world. Some of which are listed:

  1. Screenrant
  2. Fandom
  3. IMDB
These websites helped me for making some new designs and think upon some creative website which consists of most of the components in the given websites and even some new merger of them.

Idea:

I have always been mesmerized by the Web series world and how it engrasps the user and fly them to a place full of imagination, wonder and fantasy. Thus, it was deemed for me to take upon the my hobby and try to collaborate it with my passion and give upon something which actually counts.


About the Website:

This website, for now, gives the user of the latest news and reportage about numerous TV-Shows and web series, covering almost all genres which exists:

  • Action.
  • Sit-Coms.
  • Muder-Mystery.
  • War.
  • Anime/Shounen World.
  • Drama.
  • Romance.
  • Sci-Fi.
  • Fanstasy and many others.

The future plans for this website is to make this a community, where all the members of our community could interact with each other, have dicussions, forums and rant about their favourite TV-Shows.
